opennextjs-cloudflare <command>

Commands:
  opennextjs-cloudflare build [args..]    Build an OpenNext Cloudflare worker
  opennextjs-cloudflare preview [args..]  Preview a built OpenNext app with a
                                          Wrangler dev server
  opennextjs-cloudflare deploy [args..]   Deploy a built OpenNext app to
                                          Cloudflare Workers
  opennextjs-cloudflare upload [args..]   Upload a built OpenNext app to
                                          Cloudflare Workers
  opennextjs-cloudflare populateCache     Populate the cache for a built Next.js
                                          app
  opennextjs-cloudflare migrate           Set up the OpenNext Cloudflare adapter
                                          in an existing Next.js project

Options:
  -h, --help     Show help                                             [boolean]
  -v, --version  Show version number                                   [boolean]
